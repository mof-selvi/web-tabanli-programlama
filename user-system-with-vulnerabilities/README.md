# NoteApp - Güvenlik Açığı Düzeltme

Bu projede 5 adet ciddi güvenlik açığı vardır. Bu açıkları bulup düzeltmeniz gerekmektedir.

Düzeltilmiş projenize ait kodlarınızı kendinize ait bir `Readme.md` dosyası ile birlikte zip'lemeden yükleyiniz.

`Readme.md` dosyanızda, bulduğunuz her güvenlik açığını ve bu açıkları nasıl düzelttiğinizi belirtmelisiniz.

Not: Bu proje Sqlite veritabanı kullanmaktadır. Farklı bir veritabanı kullanmayınız. `install.php` dosyasını çalıştırarak veritabanını kurabilirsiniz. Kurulum sonrasın veritabanı dosyası proje dizininde oluşturulacaktır. Bu da bir güvenlik açığıdır fakat çözmeniz beklenen 5 adet güvenlik açığına dahil değildir.
